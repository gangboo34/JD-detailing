# JD Detailing Website

Free GitHub Pages website for jddetailing.biz.

## Booking

The Book Now buttons are connected to Square Appointments:

https://app.squareup.com/appointments/book/g1cyfay6eoi286/L6CY3M89W4GS8/start

## Publish

1. Upload all files to the root of a public GitHub repository.
2. Go to Settings > Pages.
3. Source: Deploy from a branch.
4. Branch: main, folder: / root.
5. Save.

## Custom Domain

The CNAME file is set to jddetailing.biz.
